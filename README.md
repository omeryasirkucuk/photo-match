## 🎯 Nedir?

**Photo Match**, kendi fotoğraflarınla oynayabileceğin retro pixel-art tarzında bir **hafıza kartı eşleştirme** oyunudur. Kartları çevir, fotoğraf çiftlerini bul, en az hamleyle bitir!

## ✨ Özellikler

| Özellik | Detay |
|---------|-------|
| 📸 **Gerçek Fotoğraflar** | 8 özel fotoğrafla oyna |
| 🎮 **4x4 Grid** | 16 kart, 8 çift |
| 🎵 **Retro Ses Efektleri** | Web Audio API ile pixel-tarzı sesler |
| 🔥 **Combo Sistemi** | Üst üste eşleştirmelerle combo yap! |
| 🎊 **Confetti Animasyonu** | Kazandığında kutlama efekti |
| 📱 **Responsive** | Mobil, tablet ve masaüstü uyumlu |
| ⚡ **Sıfır Bağımlılık** | Framework yok, build yok, tek `index.html` |

## 🎮 Nasıl Oynanır?

```
1️⃣  "OYUNU BASLAT" butonuna tıkla
2️⃣  Bir karta tıkla → kart döner, fotoğraf görünür
3️⃣  İkinci kartı tıkla → eşleşme kontrolü
4️⃣  Aynı fotoğrafsa → kartlar açık kalır ✅
5️⃣  Farklıysa → kartlar geri döner ❌
6️⃣  Tüm 8 çifti bul → kazandın! 🎊
```

> 💡 **İpucu:** Üst üste eşleştirme yaparak **COMBO** kazan!
